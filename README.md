# 🛒 Amazon Price Tracker

This Python script monitors the price of an Amazon product and sends an email alert when the price drops below the target price.

## 🚀 Features

- Web scraping using BeautifulSoup  
- Automatic price monitoring  
- Email notification  
- Secure environment variables  

## 🛠️ Tech Stack

- Python  
- Requests  
- BeautifulSoup  
- SMTP  
- python-dotenv  

## ⚙️ Setup

pip install -r requirements.txt  
python main.py
